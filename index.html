POPPEG

POPPEG is a platform that allows users to create physical photo and video albums that can later be shared as a gift

POPPEG

What is it about?
POPPEG is a platform that allows users to create physical photo and video albums that can later be shared as a gift. Living in a vibrant digital world, we have many memories stored within our phones, in the form of videos and photos. However, there are not many ways for us to share these memories with our loved ones, especially as a sincere and meaningful gift. With this problem in mind, we created this platform to allow users to pair videos with physically printed photos, which can then be viewed through our application.
How does it work?
POPPEG was built using NodeJS, Firebase, Unity and Vuforia. It consists of two components, a web application and a Unity application. The web application allows the user to upload their videos and photos. On the other hand, the Unity application allows the user to view videos on top of printed photos. The combination of these two components creates an intuitive platform that allows users to create physical photo albums that can view videos displayed on top of them. The videos are displayed using Vuforia’s Augmented Reality, creating a fulfilling experience for the user.

ARchitech

ARchitech is an augmented reality maze builder game, originally created as a fun game for users to play for orientation

ARchitech

What is it about?
ARchitech is an augmented reality maze builder game as a project for NUS Orbital 2018, Apollo. The project was originally created as a fun game for users to play for School of Computing’s orientation. It was intended to help new freshman experience interesting technology while bonding with their fellow peers.
How does it work?
ARchitech was built using Unity, Firebase and Vuforia. Users can create a maze which is superimposed on top of a dollar note. The augmented reality feature allows them to view different angles of the maze easily, helping the user to build a complex 3D maze. The maze can then be saved through our cloud database and other users will be able to attempt the maze.

HeadHunters

HeadHunters is a fun face filter application, built using Unity and ARKit’s Facial Recognition

HeadHunters

What is it about?
HeadHunters is a fun face filter application created for Clear Hackathon 2018. This project was built using Unity and ARKit’s Facial Recognition. It aimed to increase awareness for Clear Shampoo in a fun and interactive way, specifically targeting millennials. During the development of the project, we worked closely with the Unilever business team to create the ideal prototype that will suit their use case.
How does it work?
Using the application, users can display digital filters on their hair and facial features. Similar to the Instagram Face Filters, users will be able to see themselves with objects superimposed onto them, learning about the advantages of using the Clear Shampoo in a fun and exciting way. Users can trigger filters using their facial movements like a smile or a blink. The application comes with three main features, namely the Dandruff filter, Sunglasses Blink filter and the Sakura Smile filter.
